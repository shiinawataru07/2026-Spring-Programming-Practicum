####  第一次作业（类与对象作业1）

### 1A.编程填空：学生信息处理程序

这是一个 C++ 编程填空题，旨在练习**类（Class）**的基本使用、**封装（Encapsulation）**以及简单的**输入输出处理**。

题目描述

实现一个学生信息处理程序，计算一个学生的四年平均成绩。

核心要求

- 必须使用 **类（Class）** 来实现。
- 类中的所有成员变量都必须是 **私有的（private）**。
- 需要补充 `Student` 类的定义，使其能够完成输入、计算和输出功能。

输入格式

输入为一行，包含以下信息，以逗号分隔：

1.  **姓名** (字符串，可能包含空格)
2.  **年龄** (整数)
3.  **学号** (整数)
4.  **第一学年平均成绩** (整数)
5.  **第二学年平均成绩** (整数)
6.  **第三学年平均成绩** (整数)
7.  **第四学年平均成绩** (整数)

输出格式

输出一行，包含以下信息，以逗号分隔：

1.  **姓名**
2.  **年龄**
3.  **学号**
4.  **四年平均成绩** (注意：结果可能不是整数)

示例

**输入:**
```
Tom Hanks,18,7817,80,80,90,70
```

**输出:**
```
Tom Hanks,18,7817,80
```

### B: Complex类的成员函数

这是一个 C++ 编程填空题，主要考察 **运算符重载（Operator Overloading）** 和 **字符串处理（String Parsing）** 的能力。

题目描述

给定一个 `Complex`（复数）类的框架，要求补充成员函数，使得程序能够支持将一个字符串（如 `"3+4i"`）直接赋值给 `Complex` 对象。

核心要求

- **补足代码**：在 `// 在此处补充你的代码` 处编写逻辑。
- **功能实现**：实现从字符串到复数对象的赋值操作（即重载赋值运算符 `=`）。
- **限制条件**：**不能**添加新的成员变量，只能使用已有的 `r` (实部) 和 `i` (虚部)。

输入输出示例

程序不需要标准输入，直接运行 `main` 函数。

**预期输出:**
```text
3+4i
5+6i
```

代码逻辑分析

在 `main` 函数中，执行了如下操作：
```cpp
Complex a;
a = "3+4i"; // 这里触发了赋值运算符重载
a.Print();
```
你需要编写一个 `operator=` 函数，它能够：
1. 接收一个字符串（或字符指针）。
2. 解析字符串，提取出实部（`+` 号前的数字）和虚部（`+` 号后、`i` 前的数字）。
3. 将提取的数值赋值给私有成员 `r` 和 `i`。
